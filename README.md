# US Retail Sales Dashboard (2020–2024)

## Overview
An interactive Excel dashboard analysing 3,000 rows of US retail sales data 
across 5 product categories, 3 regions and 4 years.

## Features
- 6 Pivot Charts (Line, Bar, Donut, Pie)
- 4 KPI Cards with Year-on-Year metrics
- 5 Interactive Slicers (Region, Gender, Category, Sales Method, Year)
- VBA Reset Filters button
- VBA Toggle Insights button

## Tools Used
- Microsoft Excel 365
- Pivot Tables
- VBA Macros

## Key Insights
- West region consistently outperformed all other regions
- Q4 drove the highest revenue across all 4 years
- Online sales method grew year on year
  
## Problems Identified
- No unified sales tracking across regions
- Inconsistent revenue performance across quarters
- Gender and category data underutilised for targeting

## Solutions Applied
- Built dynamic dashboard with interactive slicers for instant filtering
- Created KPI cards with YoY metrics to track performance over time
- Segmented data by region, category and sales method for deeper insight

## Recommendations
- Focus marketing budget on West region — highest consistent revenue
- Investigate Q1 underperformance across all years
- Expand online sales channel — shows strongest growth trend

## How To Use
1. Download the `.xlsm` file
2. Enable macros when prompted
3. Use slicers to filter the dashboard interactively
